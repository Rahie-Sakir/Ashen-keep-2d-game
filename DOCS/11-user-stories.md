# User Stories

## Gameplay

### Story 1

**As a player**, I want responsive keyboard controls so that my actions are reflected immediately in the game.

---

### Story 2

**As a player**, I want visual updates after each interaction so that I understand the results of my actions.

---

### Story 3

**As a player**, I want the application to remain stable throughout gameplay so that I can enjoy an uninterrupted experience.

---

## Development

### Story 4

**As a developer**, I want the backend and frontend separated into different modules so that maintenance is easier.

---

### Story 5

**As a developer**, I want dedicated input handling so that controls can be modified independently.

---

### Story 6

**As a developer**, I want automated tests so that software quality can be verified after changes.

---

## Academic

### Story 7

**As an instructor**, I want well-organized documentation so that the project's design decisions are easy to evaluate.

---

### Story 8

**As a future contributor**, I want readable code and documentation so that I can understand the project quickly.
