# Ashen_keep_2d_game

## Project Name
**Ashen Keep**

## Introduction

Ashen Keep is a desktop-based adventure game developed in Python as a solo software engineering project. The project demonstrates modular software architecture by separating game logic, rendering, and input handling into independent components. This design improves maintainability, readability, and future scalability while providing an engaging gameplay experience.

The application follows a structured design where the backend manages game state and mechanics, the frontend handles rendering, and dedicated input modules process user interactions. This separation allows the project to be easily extended with new features without requiring major changes to existing code.

## Project Goals

- Develop a functional adventure game using Python.
- Demonstrate object-oriented and modular programming concepts.
- Implement clean separation between game logic and presentation.
- Create maintainable and testable source code.
- Showcase software engineering documentation and project organization.

## Scope

The current version includes:
- Game initialization and execution.
- User input handling.
- Backend game state management.
- Frontend rendering.
- Modular project structure.
- Automated testing support.

Future enhancements may include:
- Save/load functionality.
- Additional game levels.
- Expanded enemy AI.
- Audio effects.
- Multiplayer capabilities.

## Technologies Used

- Python
- Modular Python packages
- Object-Oriented Programming
