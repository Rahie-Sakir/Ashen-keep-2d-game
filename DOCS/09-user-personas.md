# User Personas

## Persona 1: Casual Player

**Age:** 20

**Background:**
Enjoys lightweight desktop games with intuitive controls.

### Goals
- Play without reading extensive instructions.
- Experience smooth gameplay.
- Receive immediate visual feedback.

### Frustrations
- Delayed controls.
- Frequent crashes.
- Confusing interfaces.

### Expectations
- Easy-to-use controls.
- Stable performance.
- Consistent rendering.

---

## Persona 2: Computer Science Student

**Age:** 22

**Background:**
Interested in studying software architecture and Python programming.

### Goals
- Understand modular game development.
- Learn project organization.
- Explore backend/frontend separation.

### Frustrations
- Poor documentation.
- Monolithic codebases.
- Difficult debugging.

### Expectations
- Clean project structure.
- Readable source code.
- Comprehensive documentation.

---

## Persona 3: Course Evaluator

### Goals
- Assess engineering practices.
- Review documentation quality.
- Verify software organization.

### Expectations
- Professional repository.
- Complete documentation.
- Logical project structure.
