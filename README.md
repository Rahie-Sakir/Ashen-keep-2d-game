# Ashen Keep: The Long Dawn

A dependency-free 2D action platformer built with Python and Tkinter. The
game uses a side-view camera, platforming, deliberate melee combat, projectile
reflection, and a timing-based parry system.

Its visual language is built from procedural silhouettes: black ink
architecture, porcelain masks, and ember-orange combat accents. No image or
audio packages are required.
